===========================================================================
                       BUILT-IN PROGRAMS REFERENCE
===========================================================================

Almost all programs live in the BIN and PLE folders and can be started from
anywhere by typing their name.

SYSTEM
-------------------------------------------
  SETUP.BIN
    First-boot wizard. Sets up your username, password
    and theme. Run it again any time to reconfigure.

  SETTINGS.BIN
    Change system settings like the timezone.

  AUTOEXEC.BIN
    Runs automatically after you log in.
    Edit it to customize what happens at startup.

  THEME.BIN
    Switch between color themes.

  FONT.BIN
    Change the terminal font. Fonts are stored in
    the FONTS folder.

  LAUNCH.BIN
    Program launcher with a graphical menu.

  FDISK.BIN
    Disk utility. View partition information.

PRODUCTIVITY
-------------------------------------------
  WRITER.BIN
    Full-screen text editor for text files.

  ED.BIN
    Line-based text editor, inspired by UNIX ed.
    See EDMAN.TXT for the manual.

  CALC.BIN
    Calculator with +, -, * and / operations.

  PROCENTC.BIN
    Quick percentage calculator.

  CALENDAR.BIN
    Shows a calendar for the current month.

  BCHART.BIN
    Creates simple bar charts.

UTILITIES
-------------------------------------------
  FETCH.BIN
    Shows system info in a stylish way.

  TREE.BIN
    Shows the folder structure as a tree.

  PS.BIN
    Lists running background processes.

  KILL.BIN
    Stops a background process by its ID.
    Use PS.BIN to find the process ID first.

  HEAD.BIN
    Shows the first lines of a text file.
    Usage: HEAD <file>

  TAIL.BIN
    Shows the last lines of a text file.
    Usage: TAIL <file>

  GREP.BIN
    Searches for text inside files.
    Usage: GREP <pattern> <file>

  DLIST.BIN
    Lists available disk drives.

  PRINT.BIN
    Sends a file to the printer (LPT port).

  MEMORY.BIN
    Lets you browse RAM contents in hex.
    Navigate with arrow keys.

  HEXEDIT.BIN
    Edit any file byte-by-byte in hex.
    For advanced users.

  CHARS.BIN
    Shows the full ASCII character table.

  CLOCK.BIN
    Displays a live clock on screen.
    Uses your timezone from TIMEZONE.CFG.

MEDIA
-------------------------------------------
  IMFPLAY.BIN
    Plays .IMF music files (AdLib format).
    Requires AdLib/OPL2 compatible sound card.
    Usage: IMFPLAY MUSIC/SONG.IMF

  WAVPLAY.BIN
    Plays .WAV sound files.
    Usage: WAVPLAY <file>

  PAINT.BIN
    A drawing program. Create and save BMP images.
    Mouse required. Supports 256 colors.

  PIANO.BIN
    Play music using your keyboard keys.
    No sound card needed (uses PC speaker).

GAMES
-------------------------------------------
  MINE.BIN
    Minesweeper. Mouse or keyboard controls.

  TETRIS.BIN
    Classic Tetris.

  TETRIS2.BIN
    Alternative Tetris version.

  SNAKE.BIN
    Guide the snake with arrow keys. Eat food,
    grow longer, and do not hit the walls.

  SPACE.BIN
    Space arcade game.

  PONG.BIN
    Classic Pong.

  MANDEL.BIN
    Mandelbrot fractal viewer.

  EYE.BIN

  BRAINF.BIN
    Brainfuck language interpreter.

  CREDITS.BIN
    Project contributors and thanks.

MS-DOS COMPATIBLE PROGRAMS
-------------------------------------------
x16-PRos can run simple MS-DOS .COM and .EXE files.
They live in the COM and EXE directories:

  HELLO.COM     "Hello, world" demo
  FRACTAL.COM   Fractal pattern generator
  CLOCK.COM     Simple clock display
  HELLO.EXE     "Hello, world" (MZ EXE format)

PLE PROGRAMS (MULTITASKING SUPPORT)
-------------------------------------------
.PLE files are large executables that supports multitasking and 
can be run in the background using the BG command. They live in the PLE
directory:

  HELLO.PLE     "Hello, world" PLE demo
  CLOCK.PLE     Background clock

  To see running tasks: PS
  To stop a task:       KILL <id>

HOW TO RUN PROGRAMS
-------------------------------------------
  Type the name:         CALC
  Or with extension:     CALC.BIN
  With an argument:      IMFPLAY MUSIC/SONG.IMF
  In the background:     bg PLE/CLOCK.PLE

Programs in BIN/ and PLE/ work from any directory.
Other programs work only in their own dir.
