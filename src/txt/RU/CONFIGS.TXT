===========================================================================
                     СПРАВОЧНИК ФАЙЛОВ КОНФИГУРАЦИИ
===========================================================================

Все файлы конфигурации находятся в каталоге CONF/ и имеют расширение
.CFG. Система читает их при запуске.

Большинство из них можно редактировать с помощью WRITER или команды
WRITE. Для некоторых есть отдельные программы для их изменения
(отмечено ниже).

SYSTEM.CFG - Настройки загрузки
-------------------------------------------
  Управляет тем, что происходит при старте системы.
  Формат: одна настройка на строку, КЛЮЧ=ЗНАЧЕНИЕ.

  LOGO=<путь>
    Путь к BMP-изображению, показываемому при загрузке.
    Пример: LOGO=BMP/LOGO.BMP

  LOGO_STRETCH=TRUE или FALSE
    Растянуть логотип загрузки на весь экран.

  START_SOUND=TRUE или FALSE
    Проигрывать короткую мелодию при старте системы.

USER.CFG - Ваше имя пользователя
-------------------------------------------
  Хранит ваше имя пользователя (до 31 символа).
  Имя появляется в командной строке.

  Чтобы изменить его, запустите SETUP или отредактируйте
  через writer.

PASSWORD.CFG - Пароль для входа
-------------------------------------------
  Хранит ваш пароль в зашифрованном виде (XOR).

  Чтобы изменить его, запустите SETUP.
  ВНИМАНИЕ: Не редактируйте этот файл вручную!
  Ручное редактирование повредит пароль.

PROMPT.CFG - Стиль командной строки
-------------------------------------------
  Определяет, как выглядит командная строка.
  Максимальная длина: 63 символа.

  Специальные переменные:
    $user   Ваше имя пользователя (из USER.CFG)
    %XX     Любой символ ASCII в шестнадцатеричном виде
            Пример: %3E - это символ ">"

THEME.CFG - Цвета терминала
-------------------------------------------
  Задаёт 16-цветную палитру терминала.
  Формат: 16 строк, один цвет на строку в виде R G B.
  Значения в диапазоне от 0 до 63.

  Порядок цветов:
    1. Чёрный          9. Тёмно-серый
    2. Синий          10. Светло-синий
    3. Зелёный        11. Светло-зелёный
    4. Голубой        12. Светло-голубой
    5. Красный        13. Светло-красный
    6. Пурпурный      14. Светло-пурпурный
    7. Коричневый     15. Жёлтый
    8. Светло-серый   16. Белый

  Пример строки: 63 0 0  (ярко-красный)

  Проще всего изменить: запустить программу THEME.BIN.

TIMEZONE.CFG - Часовой пояс
-------------------------------------------
  Одно число от -12 до 12.
  Это смещение прибавляется к UTC при показе времени.

  Пример: -5 означает UTC-5 (Восточное время США).

  Меняется программой SETTINGS.BIN или редактированием
  файла напрямую.

FONT.CFG - Шрифт терминала
-------------------------------------------
  Содержит имя файла активного шрифта из папки FONTS.

  Пример: BOLD.FNT

  Меняется программой FONT.BIN или редактированием
  файла напрямую.

FIRST_B.CFG - Флаг первой загрузки
-------------------------------------------
  Содержит одну цифру:
    1 = первая загрузка (SETUP запустится автоматически)
    0 = обычная загрузка

  Пожалуйста, не удаляйте этот файл.


Если вы удалите файл конфигурации, система вернётся
к значениям по умолчанию.
