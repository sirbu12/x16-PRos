===========================================================================
                     CONFIGURATION FILES REFERENCE
===========================================================================

All config files live in the CONF/ directory and have the .CFG
extension. The system reads them at startup.

You can edit most of them with WRITER or with the WRITE
command. Some have dedicated programs for changing them
(noted below).

SYSTEM.CFG - Boot settings
-------------------------------------------
  Controls what happens when the system starts.
  Format: one setting per line, KEY=VALUE.

  LOGO=<path>
    Path to a BMP image shown at boot.
    Example: LOGO=BMP/LOGO.BMP

  LOGO_STRETCH=TRUE or FALSE
    Stretch the boot logo to fill the screen.

  START_SOUND=TRUE or FALSE
    Play a short melody when the system starts.

USER.CFG - Your username
-------------------------------------------
  Stores your username (up to 31 characters).
  The name appears in the command prompt.

  To change it, run SETUP or edit using writer.

PASSWORD.CFG - Login password
-------------------------------------------
  Stores your password in encrypted form (XOR).

  To change it, run SETUP.
  WARNING: Do not edit this file by hand!
  Manual editing will corrupt the password.

PROMPT.CFG - Command prompt style
-------------------------------------------
  Defines how the command prompt looks.
  Max length: 63 characters.

  Special variables:
    $user   Your username (from USER.CFG)
    %XX     Any ASCII character in hex
            Example: %3E is the ">" symbol

THEME.CFG - Terminal colors
-------------------------------------------
  Sets the 16-color palette for the terminal.
  Format: 16 lines, one color per line as R G B.
  Values range from 0 to 63.

  Color order:
    1. Black          9. Dark Gray
    2. Blue          10. Light Blue
    3. Green         11. Light Green
    4. Cyan          12. Light Cyan
    5. Red           13. Light Red
    6. Magenta       14. Light Magenta
    7. Brown         15. Yellow
    8. Light Gray    16. White

  Example line: 63 0 0  (bright red)

  Easiest way to change: run THEME.BIN program.

TIMEZONE.CFG - Time zone
-------------------------------------------
  A single number from -12 to 12.
  This offset is added to UTC when showing time.

  Example: -5 means UTC-5 (US Eastern).

  Change it with SETTINGS.BIN program or edit the file directly.

FONT.CFG - Terminal font
-------------------------------------------
  Contains the filename of the active font from
  the FONTS folder.

  Example: BOLD.FNT

  Change it with FONT.BIN program or edit the file directly.

FIRST_B.CFG - First boot flag
-------------------------------------------
  Contains a single digit:
    1 = first boot (SETUP will run automatically)
    0 = normal boot

  Do not delete this file pls.


If you delete a config file, the system will fall back
to default values.
