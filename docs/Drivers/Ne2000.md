## ne2000_init

Initializes the NE2000 network card and configures:

* Hardware reset
* DMA settings
* RX ring buffer
* MAC address
* RX/TX engines

### Arguments

None.

### Returns

| Flag | Meaning                 |
| ---- | ----------------------- |
| CF=0 | Success                 |
| CF=1 | NIC not found or failed |

### Example

```asm
call ne2000_init
jc init_failed
```

---

## ne2000_get_mac

Retrieves the NIC MAC address.

### Arguments

| Register | Description                     |
| -------- | ------------------------------- |
| ES:DI    | Pointer to 6-byte output buffer |

### Returns

Writes the MAC address to the buffer.

### Example

```asm
mac_buffer rb 6

mov di, mac_buffer
call ne2000_get_mac
```

---

## ne2000_wait_isr

Waits until a specific interrupt status bit becomes set.

### Arguments

| Register | Description              |
| -------- | ------------------------ |
| AL       | ISR bit mask to wait for |

### Returns

| Flag | Meaning        |
| ---- | -------------- |
| CF=0 | Event occurred |
| CF=1 | Timeout        |

### Example

```asm
mov al, NE_ISR_RDC
call ne2000_wait_isr
jc timeout
```

---

## ne2000_send

Sends an Ethernet frame.

Frames smaller than 60 bytes are automatically padded.

### Arguments

| Register | Description            |
| -------- | ---------------------- |
| DS:SI    | Pointer to packet data |
| CX       | Packet length          |

### Returns

| Flag | Meaning            |
| ---- | ------------------ |
| CF=0 | Packet transmitted |

### Example

```asm
mov si, packet
mov cx, packet_size

call ne2000_send
```

---

## ne2000_poll

Checks for a received packet and copies it into a buffer.

### Arguments

| Register | Description               |
| -------- | ------------------------- |
| ES:DI    | Destination packet buffer |

### Returns

| Flag | Meaning             |
| ---- | ------------------- |
| CF=0 | Packet received     |
| CF=1 | No packet available |

### Example

```asm
recv_buffer rb 1600

mov di, recv_buffer
call ne2000_poll
jc no_packet

; Packet available
```
